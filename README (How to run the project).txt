How to run the React Frontend Project
1. In Github, click on code and download the zip on desktop
2. Extract the zip on your desktop
3. Open the folder using VSCode (Simply open the folder on desktop and then press Shift+Right Click and then click on Open with Code)
4. In VSCode, on the upper left you will see three dots click on it and then click on terminal and then click on new terminal
5. In the terminal, at the bottom, type npm install to install the node_modules related to project and after that type npm start to run the project
